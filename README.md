## Anon chat app

This app is developed to make chatting much private and easy without stealing your data!!

### Languages/ Tools: 
    - React
    - JavaScript
    - Nodejs
    - Mongodb
    - React Redux
    - SocketIo
    - Sawo
    - Tailwindcss
